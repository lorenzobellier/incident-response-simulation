# Incident Response Simulation

This project simulates detecting and responding to a security incident on a Linux server.

## Features
- Analyze auth logs
- Detect suspicious login attempts
- Lock compromised accounts
- Document incidents professionally

## Tools
- Ubuntu Server
- Bash scripts
- Linux logs

## Learning goals
- Understand incident detection and response workflow
- Practice log analysis
- Automate mitigation steps
